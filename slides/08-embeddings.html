O que são Embeddings?

Imagine que cada palavra ou texto pode ser representado por uma lista de números. Esses números capturam o significado do texto.

"Embeddings são como coordenadas num mapa de significados."

Exemplo visual

Texto Vetor (simplificado)
"gato" [0.2, 0.8, 0.1]
"cachorro" [0.3, 0.7, 0.2]
"carro" [0.9, 0.1, 0.3]

"gato" e "cachorro" são mais próximos! "carro" está longe.

Similaridade por Cosseno

Conexão com Geometria Analítica!

Você já estudou ângulos entre vetores? A similaridade por cosseno usa exatamente isso.

Fórmula:

\[\cos(\theta) = \frac{\vec{A} \cdot \vec{B}}{|\vec{A}| \cdot |\vec{B}|}\]

Interpretação:

  • ângulo = 0° → vetores iguais → cosseno = 1 → muito similares
  • ângulo = 90° → vetores perpendiculares → cosseno = 0 → nada similares
  • ângulo = 180° → vetores opostos → cosseno = -1 → opostos

Por que funciona?

  • Textos com significado parecido = vetores na mesma direção
  • Textos com significado diferente = vetores em direções diferentes

Por que isso importa?

Embeddings são a base para:

  • Busca semântica: encontrar textos parecidos
  • RAG: buscar documentos relevantes
  • Tradução: entender significados entre idiomas
  • Classificação: agrupar textos por tema

Prática

from sentence_transformers import SentenceTransformer

# Carregar modelo
model = SentenceTransformer('all-MiniLM-L6-v2')

# Converter textos em embeddings
textos = ["Eu amo gatos", "Eu adoro cachorros", "O carro é vermelho"]
embeddings = model.encode(textos)

# Calcular similaridade
from numpy import dot
from numpy.linalg import norm

cos_sim = lambda a, b: dot(a, b) / (norm(a) * norm(b))

Resultado esperado:

print(f"gatos vs cachorros: {cos_sim(embeddings[0], embeddings[1]):.3f}")
# ~0.7 (parecidos)

print(f"gatos vs carro: {cos_sim(embeddings[0], embeddings[2]):.3f}")
# ~0.1 (diferentes)

Para lembrar

  1. Embeddings = texto transformado em números
  2. Similaridade por cosseno = ângulo entre vetores
  3. Textos parecidos = vetores na mesma direção
  4. Conexão com geometria analítica no seu curso
