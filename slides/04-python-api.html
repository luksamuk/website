O que é uma API?

  • API = Application Programming Interface
  • Forma de programas "conversarem"
  • Você envia uma requisição, recebe uma resposta

Analogia: API é como um garçom. Você pede, ele leva pra cozinha, traz a resposta.

Vamos usar Gemini e HuggingFace no Colab!

Opção 1: Google Gemini

  • 1 milhão de tokens/dia GRATUITO
  • Precisa de conta Google
  • API key: aistudio.google.com
from google import genai
from google.genai import types
import os

# Novo SDK google-genai: usar Client
client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

# Chamada simples
response = client.models.generate_content(
    model="gemini-2.0-flash",
    contents="Olá!"
)
print(response.text)

Como obter API key do Gemini

  1. Acesse: aistudio.google.com
  2. Faça login com sua conta Google
  3. Clique em "Get API Key"
  4. Crie uma nova chave
  5. Copie e guarde bem!

API key é SECRETA. Não compartilhe!

Opção 2: HuggingFace

  • Gratuito com rate limiting
  • Vários modelos disponíveis
  • Precisa de conta no HuggingFace (gratuita)
from huggingface_hub import InferenceClient
import os

client = InferenceClient(
    model="Qwen/Qwen2.5-7B-Instruct",
    token=os.environ["HF_TOKEN"]
)

response = client.text_generation("Olá!")
print(response)

Como obter token do HuggingFace

  1. Acesse: huggingface.co
  2. Crie uma conta gratuita
  3. Settings → Access Tokens → New token
  4. Copie o token

Token é diferente de API key!

Opção 3: OpenRouter

  • Agregador de modelos de múltiplos provedores
  • Modelos gratuitos disponíveis
  • Crédito inicial gratuito
  • Integração com LangChain

Modelos gratuitos populares:

Modelo Provedor Uso
  -----------------------
  qwen/qwen3.6-plus:free Qwen Chat geral
  meta-llama/llama-3.2-3b-instruct:free Meta Rápido 
from langchain_openrouter import ChatOpenRouter

llm = ChatOpenRouter(
    model="qwen/qwen3.6-plus:free",
    temperature=0.8,
)

response = llm.invoke("Olá! Qual é a capital do Brasil?")
print(response.content)

Como usar OpenRouter

  1. Acesse: openrouter.ai
  2. Crie uma conta
  3. Gere sua API key
  4. Configure: OPENROUTER_API_KEY

Links úteis:

  • Modelos gratuitos: openrouter.ai/collections/free-models
  • Docs LangChain: docs.langchain.com/oss/python/integrations/chat/openrouter

System Instruction (Gemini)

from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

# Configurar com system instruction
response = client.models.generate_content(
    model="gemini-2.0-flash",
    contents="O que é uma lista?",
    config=types.GenerateContentConfig(
        system_instruction="Você é um tutor de Python. Responda de forma didática."
    )
)
print(response.text)

Parâmetros Importantes

  • temperature: 0.0 (determinístico) a 2.0 (criativo)
  • maxoutputtokens: Limite de tokens na resposta
# Exemplo com Gemini (novo SDK)
from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

response = client.models.generate_content(
    model="gemini-2.0-flash",
    contents="Conte uma piada!",
    config=types.GenerateContentConfig(
        temperature=0.7,
        max_output_tokens=100
    )
)
print(response.text)

Chat com Histórico (Gemini)

from google import genai
from google.genai import types

client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

# Criar chat
chat = client.chats.create(model="gemini-2.0-flash")

# Enviar mensagens
response1 = chat.send_message("Olá!")
print(response1.text)

response2 = chat.send_message("Qual é a capital do Brasil?")
print(response2.text)

Tratamento de Erros

from google import genai
from google.genai import types
from google.api_core.exceptions import NotFound, PermissionDenied

client = genai.Client(api_key=os.environ["GEMINI_API_KEY"])

try:
    response = client.models.generate_content(
        model="gemini-2.0-flash",
        contents="Olá!"
    )
    print(response.text)
except PermissionDenied:
    print("API key inválida!")
except NotFound:
    print("Modelo não encontrado!")
except Exception as e:
    print(f"Erro: {e}")
