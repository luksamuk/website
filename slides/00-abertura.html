    • openrouter.ai
    • Crédito inicial gratuito
    • Modelos gratuitos (:free)

Teste online antes de programar!

Ollama

  • Roda modelos localmente na sua máquina
  • Vamos usar o Qwen3.5:4B hoje
  • Sem internet, sem API, sem custo
ollama run qwen3.5:4b

kitty icat ./japanese.png && \
ollama run qwen3.5:4b "Transcreva o texto e depois traduza o texto para Português: ./japanese.jpg" --think=false

Open WebUI

  • Interface web local customizável
  • Conecta seus modelos e chaves de API
  • Roda localmente (para mim!)

No meu caso, uso apenas para Ollama.

LM Studio

  • Interface gráfica para modelos locais
  • Vamos ver o Gemma 4 E2B (roda em celular!)
  • Alternativa fácil para quem não quer terminal
  • Site: https://lmstudio.ai/

ask-ai no Terminal

  • Ferramenta que desenvolvi para uso pessoal
  • Consulta múltiplas fontes de IA
  • Interface simples no terminal
ask ocr ./japanese.jpg | ask translate :br

Vocês vão construir algo ligeiramente similar ao longo do curso.
