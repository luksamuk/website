Prompting Básico


Como falar com IA


Guilda de IA

O Problema

  • A mesma pergunta pode gerar respostas diferentes
  • Dependendo de como você formula

Exemplo

Vago: 

"Me fale sobre gatos"
→ resposta genérica

Específico: 

"Quais as 3 raças de gatos mais populares no Brasil?"
→ resposta específica

O que é prompt?

  • Prompt = instrução que você dá para o modelo
  • A pergunta, o comando, o contexto
  • Tudo junto

O prompt é como uma receita. Ingredientes claros = prato previsível.

Técnicas básicas

1. Zero-shot

Você pede algo sem dar exemplos. 

Traduza para inglês: "Bom dia"

O modelo nunca viu você traduzir antes, mas tenta fazer.

2. Few-shot

Você dá exemplos antes de pedir. 

Traduza para inglês:
"Gato" → "Cat"
"Cachorro" → "Dog"
"Pássaro" → ?

O modelo aprende o padrão pelos exemplos.

Por que few-shot funciona?

  • O modelo vê o que você quer
  • Segue o molde
  • Reduz ambiguidade

Exemplos são como mostrar em vez de explicar.

Framework R-C-E-F

Uma estrutura útil para prompts:

Letra Significado Exemplo
R Role (papel) "Você é um professor de matemática"
C Context "O aluno está no ensino médio"
E Examples "Explique como: 2x + 3 = 7 → x = 2"
F Format "Responda em bullets"

Exemplo completo

R: Você é um tutor de matemática paciente.
C: O aluno tem dificuldade com equações.
E: Para 2x + 4 = 10: subtraia 4 dos dois lados,
   depois divida por 2.
F: Responda passo a passo, numerando.

Pergunta: Como resolvo 3x + 6 = 21?

Resultado comparado

Sem R-C-E-F:

"3x + 6 = 21, x = 5"

Com R-C-E-F:

"Vamos resolver passo a passo:

  1. Subtraia 6 de ambos os lados: 3x = 15
  2. Divida por 3: x = 5
  3. Verificação: 3 × 5 + 6 = 21 ✓"

Alucinações vs Prompt ruim

Às vezes o modelo erra. Por quê?

  1. Hallucinação → modelo inventa (limitação)
  2. Prompt ruim → você não deixou claro

Como distinguir

  • Refaça o prompt com mais contexto
  • Se ainda errar, é alucinação
  • Se melhorar, era prompt ruim

Dicas práticas

✅ Bom:

"Liste as 5 capitais mais populosas do Brasil
em ordem decrescente"

"Explique recursão como se eu tivesse 12 anos"

"Corrija o texto abaixo mantendo o estilo do autor"

❌ Ruim:

"Me fale algo" (vago demais)

"O que você acha?" (subjetivo)

"Responda sim ou não: o número 7 é legal?" (não factual)

Para lembrar

"O prompt é como uma receita. Ingredientes claros = prato previsível."

Quanto mais específico você for, mais próxima a resposta do que você espera.
