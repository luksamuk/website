LLM: Large Language Model

LLM = Modelo de Linguagem Grande

"Um LLM é como um bibliotecário que leu toda a internet e tenta adivinhar a próxima palavra."

Não é mágica. É probabilidade.

Como funciona (simplificado)

  1. Tokenização: texto vira números
  2. Embeddings: cada palavra vira um vetor
  3. Attention: o modelo "presta atenção" nas relações
  4. Predição: qual a próxima palavra mais provável?
Input: "O céu é"

[processa bilhões de parâmetros]

Output: "azul" (com 94% de probabilidade)

LLM vs Machine Learning Tradicional

Machine Learning Clássico LLM (Generativo)
Classifica dados Gera conteúdo
Treinado para UMA tarefa Treinado para TUDO
Precisa de dados rotulados Aprende sozinho
Explicável Caixa preta

"ML clássico: detector de spam\nLLM: escreve um email"

Alucinações: A IA inventa coisas

Demo ao vivo

Vamos perguntar sobre algo que não existe: 

"Quem é João da Silva Pereira Costa,
o inventor do motor a vapor brasileiro?"

O que acontece?

  • A IA vai responder
  • Com detalhes convincentes
  • Mas a pessoa não existe

"Alucinação = respostas confiantes, mas inventadas"

Por que isso acontece?

LLMs não sabem "verdade" — sabem probabilidade.

  • Treinado para completar texto
  • Não tem acesso à realidade
  • "João Costa" parece uma resposta plausível

"LLMs não mentem. Eles confabulam."

Limites da IA

A IA NÃO sabe:

  • O que é verdade ou mentira
  • Seu contexto (a menos que você fale)
  • Coisas que não estavam no treino
  • Matemática complexa (confia na probabilidade)

"IA é um estudante que leu tudo, mas não entende tudo."

Como testar?

Sempre verifique:

  • Fontes confiáveis
  • Cross-check com outras fontes
  • Se importa se está certo

"IA é ferramenta, não oráculo."
