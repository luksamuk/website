Objetivo da semana

Ter um agente funcionando com:

  • Personalidade definida
  • Uma ferramenta funcional
  • Memória de conversa
  • (Opcional) RAG implementado

Checklist básico

Personalidade:

  • [ ] O agente tem um nome?
  • [ ] O tom de voz está definido?
  • [ ] As instruções de sistema estão claras?

Ferramenta:

  • [ ] A ferramenta funciona sozinha?
  • [ ] O agente sabe quando usar?
  • [ ] A resposta chega correta?

Memória:

  • [ ] O agente lembra da conversa?
  • [ ] O histórico não está inflando demais?

RAG (opcional):

  • [ ] Embeddings funcionando?
  • [ ] Busca retorna documentos relevantes?
  • [ ] Resposta usa o contexto?

Demo ask-ai

Nessa aula vamos demonstrar um agente completo com:

  • Múltiplas ferramentas
  • RAG funcionando
  • Personalidade coerente

Isso é expositivo — você não precisa implementar tudo. É para ver as possibilidades e se inspirar.

Formato da aula

Primeira metade: Demo e mentoria

  • Demo de agente completo (ask-ai)
  • Dúvidas e ajustes individuais
  • Tempo para desenvolver

Segunda metade: Desenvolvimento livre

  • Cada um no seu ritmo
  • Mentoria disponível
  • Testando e ajustando

Dicas para desenvolver

Comece simples:

  1. Agente conversa básico funcionando
  2. Adicione uma ferramenta
  3. Defina personalidade
  4. Adicione memória
  5. (Só depois) RAG

Não tente fazer tudo de uma vez.

Teste sempre:

  • "Oi, quem é você?"
  • "Quanto é 2 + 2?" (se tiver calculadora)
  • "O que você sabe sobre X?" (se tiver RAG)

Para lembrar

"Um agente simples funcionando vale mais que um agente complexo quebrado."
