* O que é um agente?

#+ATTR_REVEAL: :frag (appear)
Agente = LLM + estrutura para conversar

#+ATTR_REVEAL: :frag (appear)
O LLM puro só responde uma pergunta. Um agente:
- Lembra da conversa (memória)
- Tem personalidade (instruções)
- Pode usar ferramentas (extensões)

* Componentes do agente

** 1. Instruções (System)
Define quem o agente é.

#+BEGIN_SRC python
instrucoes = "Você é um assistente educado e objetivo."
#+END_SRC

** 2. Histórico (Memory)
Guarda a conversa.

#+BEGIN_SRC python
historico = [
    {"role": "usuário", "content": "Oi!"},
    {"role": "assistente", "content": "Olá! Como posso ajudar?"}
]
#+END_SRC

** 3. Mensagem (Input)
A pergunta atual.

** 4. Resposta (Output)
A resposta do modelo.

* Por que estado separado?

#+ATTR_REVEAL: :frag (appear)
Estado em dicionário = mais simples para iniciantes.

Funções puras são mais fáceis de testar e entender.

** Com funções:

#+BEGIN_SRC python
agente = criar_agente("Você é educado.")
conversar(agente, "Oi")       # histórico atualiza
conversar(agente, "Lembra o que eu disse?")  # tem contexto
#+END_SRC

* O problema do histórico infinito

#+ATTR_REVEAL: :frag (appear)
Se a conversa crescer demais, o prompt fica gigante.

**Solução:** limitar histórico.

#+BEGIN_SRC python
# Pegar só as últimas 10 trocas
for msg in agente["historico"][-10:]:
    ...
#+END_SRC

#+BEGIN_SRC text
