* Agora é hora de consolidar

#+ATTR_REVEAL: :frag (appear)
Essa semana é para *você* desenvolver seu agente.

**O objetivo:** ter um agente funcionando com:

- Personalidade definida
- Uma ferramenta funcional
- Memória de conversa
- (Opcional) RAG implementado

* Checklist básico

**Personalidade:**
- [ ] O agente tem um nome?
- [ ] O tom de voz está definido?
- [ ] As instruções de sistema estão claras?

**Ferramenta:**
- [ ] A ferramenta funciona sozinha?
- [ ] O agente sabe quando usar?
- [ ] A resposta chega correta?

**Memória:**
- [ ] O agente lembra da conversa?
- [ ] O histórico não está inflando demais?

**RAG (opcional):**
- [ ] Embeddings funcionando?
- [ ] Busca retorna documentos relevantes?
- [ ] Resposta usa o contexto?
