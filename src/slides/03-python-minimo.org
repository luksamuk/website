* Listas e Dicionários

#+BEGIN_SRC python
# Lista: sequência ordenada
frutas = ["maçã", "banana", "laranja"]
frutas.append("uva")
frutas[0]    # "maçã"
frutas[-1]   # "uva"

# Dicionário: pares chave-valor
pessoa = {"nome": "João", "idade": 30}
pessoa["nome"]           # "João"
pessoa["idade"] = 31     # atualiza
pessoa["cidade"] = "BH"  # adiciona
#+END_SRC

* Funções

#+BEGIN_SRC python
def saudar(nome, saudacao="Olá"):
    return f"{saudacao}, {nome}!"

saudar("Maria")        # "Olá, Maria!"
saudar("João", "Oi")  # "Oi, João!"
#+END_SRC

#+BEGIN_SRC python

* Loops

#+BEGIN_SRC python
# Iterar em lista
for fruta in frutas:
    print(fruta)

# Range
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# While
count = 0
while count < 3:
    print(count)
    count += 1
#+END_SRC

* O que é uma API?

API = Interface de Programação de Aplicações

#+begin_quote
Você manda uma mensagem (=request=) →
O servidor processa →
Ele devolve uma resposta (=response=)
#+end_quote

Analogia: Você → Garçom (=API=) → Cozinha → Prato (resposta)

* API de LLM

Para usar um LLM via API:

1. Você manda um *prompt* (request)
2. O modelo processa
3. Devolve a *resposta* (response)

* Google Gemini (Grátis!)

- *1 milhão de tokens/dia* gratuito
- Funciona direto no Colab
- API key gratuita em =aistudio.google.com=

#+BEGIN_SRC python
# Instalar: !pip install google-genai
from google import genai
from google.genai import types
from google.colab import userdata

client = genai.Client(api_key=userdata.get('GEMINI_API_KEY'))
response = client.models.generate_content(
    model="gemini-2.5-flash-preview",
    contents="Olá! Qual é a capital do Brasil?"
)
print(response.text)
#+END_SRC

* Gemini: Chat com Memória

#+BEGIN_SRC python
chat = client.chats.create(model="gemini-2.5-flash-preview")
r1 = chat.send_message("Olá!")
print(r1.text)

r2 = chat.send_message("Qual a capital do Brasil?")
print(r2.text)  # Lembra da conversa!
#+END_SRC

* Gemini: System Prompt

#+BEGIN_SRC python
response = client.models.generate_content(
    model="gemini-2.5-flash-preview",
    contents="O que é uma lista?",
    config=types.GenerateContentConfig(
        system_instruction="Você é um tutor de Python. Responda de forma didática."
    )
)
print(response.text)
#+END_SRC
