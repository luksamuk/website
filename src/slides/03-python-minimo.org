* Por que Python?

#+ATTR_REVEAL: :frag (appear)
Python é a linguagem mais usada em IA por:

1. *Sintaxe simples* → parece pseudocódigo
2. *Bibliotecas* → milhões de pacotes prontos
3. *Comunidade* → documentação em todo lugar

* O mínimo que você precisa saber

** Variáveis
#+BEGIN_SRC python
nome = "Maria"      # texto (string)
idade = 25          # número inteiro (int)
altura = 1.70       # número decimal (float)
estudante = True    # verdadeiro/falso (bool)
#+END_SRC

** Listas
#+BEGIN_SRC python
frutas = ["maçã", "banana", "laranja"]
frutas.append("uva")       # adiciona
frutas[0]                  # "maçã" (primeiro)
frutas[-1]                 # "uva" (último)
#+END_SRC

** Funções
#+BEGIN_SRC python
def saudar(nome):
    return f"Olá, {nome}!"

saudar("João")  # "Olá, João!"
#+END_SRC

** Condicionais
#+BEGIN_SRC python
if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")
#+END_SRC

* O que é uma API?

#+ATTR_REVEAL: :frag (appear)
API = Interface de Programação de Aplicações

#+ATTR_REVEAL: :frag (appear)
#+begin_quote
Você manda uma mensagem (request)
O servidor processa
Ele devolve uma resposta (response)
#+end_quote>

** Analogia restaurante

#+ATTR_REVEAL: :frag (appear)
- Você → garçom → *API* → cozinha → resposta (prato)

* Google Colab

#+ATTR_REVEAL: :frag (appear)
Colab = Jupyter Notebook na nuvem

**Vantagens:**
- Não precisa instalar nada
- Roda no navegador
- Grátis (com limites)

** Como usar
#+ATTR_REVEAL: :frag (appear)
1. Acesse colab.research.google.com
2. Crie um novo notebook
3. Digite código em células
4. Execute com Shift+Enter
