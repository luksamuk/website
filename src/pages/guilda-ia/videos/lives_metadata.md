## Live 01 — O que é IA hoje?

### Título

[LIVE] Guilda de IA — Aula 01: O que é IA hoje?

### Descrição

Nesta aula, a gente desfaz a confusão entre IA, LLM, RAG e Agente — quatro termos que vivem sendo usados como se fossem a mesma coisa.

O que cobrimos nesta aula:
- O que é IA como campo e o que NÃO é apenas "ChatGPT"
- LLM: o que o modelo realmente faz (e os 4 superpoderes: tool calling, raciocínio, contexto longo, multimodalidade)
- RAG: como buscar informações nos seus documentos e gerar respostas fundamentadas
- Agente: LLM + ferramentas + memória — o próximo passo além do chat
- Por que o jargão importa na hora de construir soluções reais

Materiais e apostilas: https://luksamuk.codes/pages/guilda-ia
Slides: https://luksamuk.codes/slides/01-nomenclaturas.html

Pré-requisitos: Lógica básica, curiosidade e conta Google (para Google Colab).
Tudo no navegador. Zero instalação obrigatória.

## Live 03 — Python Mínimo

### Título

[LIVE] Guilda de IA — Aula 03: Python Mínimo

### Descrição

Python é a ferramenta. O LLM é o cérebro. A API é a ponte entre eles.

Nesta aula, a gente cobre o Python mínimo que você precisa pra começar a construir com IA — sem instalação, direto no navegador.

O que cobrimos nesta aula:
- Variáveis, tipos e f-strings — a base de tudo
- Listas e dicionários — como APIs devolvem dados
- Funções — blocos reutilizáveis (e como toda chamada de API é uma função)
- Condicionais e loops — If, for e while pra resolver qualquer lógica
- Demo ao vivo no Google Colab
- Fofocas da semana: ZAYA1-8B e Mythos vs GPT-5.5 Cyber

Slides: https://luksamuk.codes/slides/03-python-minimo.html

Pré-requisitos: Lógica básica, curiosidade e conta Google (para Google Colab).
Tudo no navegador. Zero instalação obrigatória.
